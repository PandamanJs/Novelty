### 11. Writing Workflow

#### Recommended Process

**1. Planning Phase**
- Use Outline Builder to structure your story
- Create chapters and scenes
- Add scene descriptions

**2. Drafting Phase**
- Write each scene in the editor
- Mark scenes as "In Progress" as you work
- Don't worry about perfection

**3. Revision Phase**
- Use AI Assistant for grammar and style
- Check plot holes with Plot Assistant
- Rewrite passages with Rewrite tool
- Mark scenes as "Completed"

**4. Review Phase**
- Read story using Timeline or Story Map view
- Check for consistency (Characters, Locations)
- Use Readability Analyzer
- Verify pacing and flow

**5. Publishing Phase**
- Export in desired format
- Set tags and metadata
- Configure publishing settings
- Export or publish

### 16. Best Practices

**Naming Conventions**
- Chapters: "Chapter 1: Title" or "Act 1, Chapter 2"
- Scenes: "Inciting Incident" or "Secret Revealed"
- Drafts: "Draft 1 - Opening Revision" or "v2.3"

**Organization**
- One section per act or major story division
- One chapter per major scene or event
- One scene per point-of-view character or setting
- Keep scenes between 500-3000 words

**Workflow**
- Save drafts after major revisions
- Mark scenes as complete when done
- Use timeline to check overall story flow
- Export for offline backup
